# Backlog

> Unified backlog. New project ideas, deferred work, experiment candidates.
> No per-person split — use `[project]` prefix instead. Use `[new]` or `[misc]` if there's no real project.
> Format: `- P{n} [{project}] {content} — {proposer} ({YYYY-MM-DD})`
> Priority: **P1** (high) / **P2** (default) / **P3** (low)
> Sub-lines (free-form notes, links, context) are optional.
