# Request Queue

> Cross-team requests (review / work / decision).
> Format: `{requester} → {receiver}: [{project}] {content} — {YYYY-MM-DD}`
> No-duplicate rule: a queued request must NOT also appear as `@today` on the receiver's workboard.
> Optional sub-lines: `📎 artifact`, `🔍 focus`, `⏰ YYYY-MM-DD` deadline, `🔄 changes-requested → @{requester} in-progress (MM-DD)`, `🔄 awaiting-rereview (MM-DD)`
